# 前言

其实这次国赛，对于我本人来说是失败的。这毋庸置疑，究其原因:

  1. 有着由于蓝桥杯省赛而导致的对国赛的轻视和半摆烂
  2. 国赛对于日常的积累考验更大，由于我的菜，和当时由于要写入党材料，一个月左右没有怎么学习前端，导致对于前端的一些东西的淡忘。(现在想想这个只占一小部分)
  3. 天时地利人和皆无，所失败。
  4. 对 jq 以及一些知识掌握的不全面 (技术栈不全被干烂了属实是)

归根结底，还是自己很菜。（文章又是半夜开始写，自己菜的睡不着，hhh）

话不多说，开始

# 第三题 水果消消乐


第三题，就主要看你的 jq 了，或者原生。但是 jq 写起来必然是更快一点。
但是我也没有学过 jq, 因为一直感觉 jq 已经被弃用了，精通也没啥用.
所以本题我会用 原生 js 和 jq 两种方式去写
首先原生的
… 我放弃了，有哪位大佬愿意用 js 原生写，写完请发我一份。。。
时间比 简直被 jq 拉爆
首先，我这里做了一些参考

  • https://blog.csdn.net/m0_51969330/article/details/125360660
    海底烧烤店 ai 大佬写的。

第一步，去分析需求

我们去看题目要求和它给的 gif 图片演示。

  1. 需要实现图片的进入和隐藏，以及按钮的隐藏
  2. 需要在点击图片的时候进行一个显示
  3. 在点击到两个图片的时候，要进行一个对比，如果不同就直接翻转下去，如果是相同的就直接把父元素的 div 标签进行一个隐藏
  4. 在答对的时候给分数加上 2 分，在答错的时候给分数减 2 分

那么来干活吧.

总结：失败的原因，对 jq 的不熟悉。一些动画效果是 jq 自带的，之前用 js 去写，还去写了动画…
评语：如果会 jq, 其实我感觉也不难。性价比是真的高。

    • 在点击 = 的时候得到结果，输出到 结果显示区域。
